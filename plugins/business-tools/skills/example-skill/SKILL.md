# Example skill

This is a placeholder so the plugin loads correctly. When invoked, confirm the
skill is working and remind the user they can now replace this file or add new
skills alongside it.

## How to add a real skill

1. Make a new folder under `skills/`, e.g. `skills/my-new-skill/`.
2. Add a `SKILL.md` inside it with `name` and `description` frontmatter (like this file).
3. Write the instructions you want Claude to follow in the body.
4. Commit the change, then click "Update" on the marketplace in Cowork to resync.
